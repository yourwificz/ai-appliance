# Contributing

Thank you for your interest in AI Appliance.

This project documents the engineering journey of building a practical, self-hosted AI appliance using enterprise hardware and open technologies.

## Ways to contribute

Contributions of all sizes are welcome, including:

- Bug reports
- Feature requests
- Performance benchmarks
- Hardware compatibility reports
- Documentation improvements
- Code contributions

## Before submitting

Please search existing issues before opening a new one.

When reporting performance or benchmark results, include as much information as possible about:

- Hardware
- Operating system
- Software versions
- Configuration
- Benchmark methodology

This helps keep comparisons meaningful and reproducible.

## Philosophy

The project values:

- Reproducibility
- Honest benchmarking
- Practical engineering
- Open technologies
- Clear documentation

Thank you for helping improve AI Appliance.
